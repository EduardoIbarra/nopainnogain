¿Cómo funciona la visita gratis?

 Esta promoción está limitada a un único evento por usuario y se aplicará de manera automática a aquellos clientes que usen la plataforma por primera vez. Cada vez que un cliente se registra en LIBERI y realiza su primera compra en cualquiera de los CAFs (Centro de Acondicionamiento Físico), se generará automáticamente un código válido para una visita de regalo útil en el mismo CAF (Centro de Acondicionamiento Físico) en el que realizó dicha compra. El código válido por una visita de regalo tiene una vigencia de 7 días naturales a partir de que se genera.

¡Cómo funciona la visita de regalo!

 Para activar esta promoción, el usuario deberá realizar 6 visitas pagadas en un mismo gimnasio. Dichas visitas pueden o no ser consecutivas. Después de generar las 6 visitas pagadas en el mismo CAF (Centro de Acondicionamiento Físico), se le generarán de manera automática 2 códigos válidos para visita gratis en el mismo establecimiento, (esta promoción sólo se generará una sola vez y será aplicable únicamente en el primer CAF (Centro de Acondicionamiento Físico) en el cual el usuario activo acumuló 6 visitas pagadas). El primero de esos 2 códigos será útil para un tercero, el cual deberá ser usuario nuevo obligatoriamente. (Usuario nuevo: aquel que nunca ha comprado una visita).
El segundo código será para el usuario activo (quien acumuló las 6 visitas) quien podrá usar su código de regalo una vez que la persona que invitó (usuario nuevo) use el código propio.
El código de regalo se lo envía el usuario activo al usuario nuevo a través de LIBERI. El uso de ambos códigos será exclusivos para el CAF (Centro de Acondicionamiento Físico) en el que el usuario activo realizó sus primeras 6 compras.
